# Vecchia Parallel Partial Emulator (VPPE)
R implementation of the scaled Vecchia approximation within the framework of parallel partial emulation (Seidman and Spiller, 2026)

## Running the VPPE
To run VPPE on your own device, first install the packages _GpGp4RG_ and _RobustGaSPV_. The packages _GpGp4RG_ and _RobustGaSPV_ are the packages _GpGp_ (Guinness, 2018) and _RobustGaSP_ (Gu, 2018), adapted for use with each other in implementing the VPPE.

We recommend running the ``VPPE_Demo.qmd`` file that reproduces Figure 1 from Seidman and Spiller (2026).

## Reference
Seidman, J., & Spiller, E. T. (2025). VPPE: Application of Scaled Vecchia Approximations to Parallel Partial Emulation. arXiv preprint arXiv:2508.19144.
